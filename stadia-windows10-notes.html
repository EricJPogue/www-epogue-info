Steps:
1 - Install Windows 10 Pro with default options... link with Microsoft ID "epogue@epogue.com".

2 - Set up local machine options... keyboard, mouse, monitor, etc. On Mac Pro disable "AMD CrossFireX" from "AMD FirePro Control Center" under "Performance" section or "Shutdown" will not work in Bootcamp.

3a - Install Visual Studio 17 Professional from Visual Studio site be sure to select C++ and Git options under the custom install options. This step (3c) should be done instead of step 3b below.

3b* - Alternative to 3a (don't do both) Set up Git to run in PowerShell... Git for Windows.

Select the following Git for Windows installation options:
- "Use Git from the Widows Command Promt"
- "Checkout as-is, commit as-is"
- "Use Windows default console window"
- Select default options otherwise

4 - Set up PowerShell to run scripts by setting execution policy to "RemoteSigned" by running PowerShell in Administrator mode and running the following commend... See Using Set-ExecutionPolicy for more information.
Set-ExecutionPolicy RemoteSigned

Restart PowerShell session to have new ExecutionPolicy take effect.

5 - Copy the following PowerShell script, past it into your PowerShell session, and execute it to create the desired development directory structure
Set-Location C:\
New-Item -ItemType Directory ".\Source"

6 - Clone EJP project
Set-Location C:\Source
git clone https://github.com/EricJPogue/EJP.git

7 - Clone www-epogue-info project
Set-Location C:\Source
git clone https://github.com/EricJPogue/www-epogue-info.git

8 - Set Git username, email, editor, and matching for commits
Git config --global user.name "EricJPogue"
Git config --global user.email "epogue@epogue.com"
Git config --global push.default simple

9 - Verify Git username and email
Git config --global user.name
Git config --global user.email
Git config --list

10 - Install Microsoft Code text editor for Windows... Code for Windows

11 - Upate "Last Utilized:" field below using Code text editor, check file into GitHub, and verify that it is updated... here

12 - Install personal PowerShell scripts and PowerShell profile
$DocumentsPath=$env:homepath + '\Documents'
set-location $DocumentsPath
git clone https://github.com/EricJPogue/Scripts.git
.\Scripts\Update-PSProfile.ps1

13 - Set up PowerShell profile and scripts. Be sure to test the "xgs" alias.
