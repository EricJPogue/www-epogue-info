Stadia Windows 10 Setup

Steps:
1 - Install Windows 10 Professional with default options... link with user Microsoft ID "epogue@epogue.com"

2 - Set up local machine options... keyboard, mouse, monitor, etc

3 - Set up PowerShell to run scripts by setting execution policy to "RemoteSigned" by running PowerShell in Administrator mode... Using the Set-ExecutionPolicy Cmdlet

4 - Set up Git to run in PowerShell... Git for Windows

Select the following Git for Windows installation options:
- "Use Git from the Widows Command Promt"
- "Checkout as-is, commit as-is"
- "Use Windows default console window"
- Select default options otherwise

5 - Create "Development" folder in the root of the default hard drive

6 - Create "Development\Source" folder

7 - Move to "Development\Source"

8 - Execute "git clone https://github.com/EricJPogue/EJP.git" from PowerShell

9 - Execute "https://github.com/EricJPogue/www-epogue-info.git" from PowerShell

10 - Install Microsoft Code text editor for Windows... Code for Windows
